Install or upgrade R and RStudio

Questions

  • Have you installed R? Do you have the latest version?
  • Have you installed RStudio? Do you have the latest version?


Note

Do you have R installed on your system and if yes is the version current?


This is a good time to make sure your R installation is current.
Check your current version by using the command below:

R.version.string
#> [1] "R version 4.2.2 (2022-10-31)"

Update your R packages:

update.packages(ask = FALSE, checkBuilt = TRUE)

You can check and install the latest pre-compiled binary of R for your OS from here
